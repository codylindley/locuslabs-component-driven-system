You shouldn't provide a Meta declaration.
You can refer to existing stories (i.e. <Story id="...">) but cannot define new stories (i.e. <Story name="...">).

### Description

Description: The modal button is a facet of the Modal primitive component. It is a styled link that the user must interact with to perform an actionButton in the modal (i.e. 'Continue' after entering login credentials in the modal).
The modal button can be configured with or without an icon and is dependent on each use case.
