### Description

The popover is an non-interactive dialog that sits on top of the applicationâ€™s main UI. It provides helpful information about using the app to the end-user and are meant to be used primarily for supplemental content and should never get in the way of critical tasks. In most cases, the popover should not contain interactive content (i.e. buttons or links). The popover is not a required component in the rendered end user application and should be able to be hidden by the developer implementing it given that definition.
