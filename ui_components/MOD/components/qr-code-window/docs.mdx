### Description

Description: The QR Code Window is an oscillating view which will appear on screen betweem the Map default map view. It is consists of three containers; headline container, a QR Code container and an Instructions container two of which will be config by the user and one {Instructions container} will be pre config. There are three variations of this component all depending on the height of the header component.
