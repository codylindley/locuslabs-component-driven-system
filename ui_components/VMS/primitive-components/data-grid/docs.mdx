You shouldn't provide a Meta declaration.
You can refer to existing stories (i.e. <Story id="...">) but cannot define new stories (i.e. <Story name="...">).

### Component Props

The vms-data-grid component wraps the third-party <code>&lt;ReactDataGrid /&gt;</code> component. (See: [Getting Started](https://reactdatagrid.io/docs/getting-started), [Documentation](https://reactdatagrid.io/docs/), [Demo](https://reactdatagrid.io/demo), [API/Props](https://reactdatagrid.io/docs/api-reference)). Any property found in the <code>&lt;ReactDataGrid /&gt;</code> [API](https://reactdatagrid.io/docs/api-reference) can be passed to the vms-data-grid component.
