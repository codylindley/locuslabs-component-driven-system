You shouldn't provide a Meta declaration.
You can refer to existing stories (i.e. <Story id="...">) but cannot define new stories (i.e. <Story name="...">).

### Description

The checkbox is a GUI widget that permits the user to make a binary choice, i.e. a choice between one of two possible mutually exclusive options. The checkbox contains by default a toggle and a text label.
