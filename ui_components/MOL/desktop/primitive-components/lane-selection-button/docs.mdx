### Description

The desktop lane selection button has 2 different use cases within the app:

1. To display other security lanes available at the same checkpoint in Security Checkpoint POI window. Tapping on one will open the corresponding Security Checkpoint POI.
2. To display security lanes as well as custom and immigration lanes when a user performs navigation to a location that takes them through a Security Checkpoint in a pop-up modal that the user must select to continue to proceed with their desired route.

Lane selection buttons either display a logo image within the button or plain text if there is no logo available.
